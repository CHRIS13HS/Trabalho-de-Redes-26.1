1. Introdução

A internet depende de protocolos de comunicação para funcionar corretamente, sendo o HTTP um dos principais responsáveis pela troca de informações entre navegadores e servidores. Com o crescimento do uso da web e o aumento das preocupações com desempenho, estabilidade e segurança, surgiram novas versões do protocolo, como o HTTP/3, que prometem melhorar a velocidade de carregamento e a experiência dos usuários.

Apesar dessas melhorias, ainda existem poucas análises práticas comparando o desempenho do HTTP/3 com versões anteriores, como o HTTP/1.1. Isso dificulta a compreensão sobre os reais benefícios do novo protocolo em diferentes cenários de uso, principalmente em relação à latência, segurança e eficiência da comunicação.

2. Proposta

A proposta deste trabalho é realizar uma avaliação comparativa entre os protocolos HTTP/3 e HTTP/1.1, analisando fatores como desempenho, tempo de resposta e estabilidade da conexão. O objetivo é identificar as principais diferenças entre as versões e verificar se o HTTP/3 realmente apresenta vantagens significativas no uso da web atual.

Para o desenvolvimento do trabalho, serão utilizadas tecnologias como a linguagem Python para criação dos testes e coleta de dados, além do Docker para simular o ambiente de experimentação. Também poderão ser utilizadas bibliotecas de rede, como socket e ferramentas de monitoramento para análise dos resultados obtidos durante os testes.
